PROJECT: QuitReady Report & Web Layout Stabilization (Final Pass)

OBJECTIVE

Implement a stable, professional layout system for QuitReady that:

• Produces a premium, full-feeling PDF report
• Produces a professional web report that mirrors the structure and styling of the PDF
• Displays correctly on desktop and mobile
• Eliminates layout drift between PDF and web
• Fixes whitespace, spacing, alignment, and typography issues
• Ensures shock scenarios behave correctly for both single and partnered users

The PDF remains the visual reference.  
The web report should follow the same section order and visual hierarchy, but be responsive and mobile friendly.

--------------------------------------------------

GLOBAL DESIGN SYSTEM

Create shared design tokens used by BOTH the PDF and web report.

Typography

Title: 30px  
Section Header: 20px  
Subheader: 16px  
Body: 13px  
Metric Numbers: 24px  
Table Header: 12px  

Font hierarchy must be consistent across the entire report.

Spacing system

XS: 6px  
SM: 10px  
MD: 16px  
LG: 28px  
XL: 44px  

Rules:

• No section should touch the next section without at least MD spacing.
• Major sections use LG spacing.
• Page headers use XL spacing from first content element.

Content container (web)

Max width: 1100px  
Centered horizontally

Mobile breakpoint

768px
